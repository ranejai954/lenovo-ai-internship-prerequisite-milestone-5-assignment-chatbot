# 🎓 Campus Buddy: Graphical AI Chatbot

Campus Buddy is a desktop-based graphical user interface (GUI) chatbot application built using **Python** and **Tkinter**. It is designed to act as an automated campus assistant, helping newly admitted students and visiting parents seamlessly find answers to recurring campus questions.

## 🚀 Features
*   **Interactive UI Dashboard:** A modern dark-themed window layout that replaces clunky command lines.
*   **Button-Driven Interaction:** Eliminates human typing errors by using explicit navigation buttons.
*   **Contextual Data Retrieval:** Instant automated answers regarding Admissions, Facilities, and Contacts.
*   **Graceful Termination Flow:** Implements secure application closure by auto-disabling controls upon exit.

## 🛠️ Technologies Used
*   **Language:** Python 3.12+
*   **GUI Framework:** Tkinter (Native Python Library)
*   **IDE Environment:** Visual Studio Code (VS Code)
